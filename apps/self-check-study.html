01.

점수가 쌓이는
학습 일기.

"했다 / 안 했다"는 너무 거칠죠. 목표 시간 대비 도달률 70% 와 만족도 이모티 30% 의 가중평균으로 세션 점수를 산출합니다. 같은 "완료"도 다르게 기록됩니다.

02.

한 달이 한 장의
풍경으로.

요일별 무지개 7색 × 점수 5단계 오퍼시티 의 월별 달력. 빈 셀이 적은 달, 진한 셀이 많은 주가 한눈에 보입니다. 꾸준함을 색으로 본다는 것.

03.

진짜 보상은
카톡에서.

앱 안의 점수보다 부모·조부모·친구의 한 마디가 더 강력합니다. 시스템 공유 시트로 카톡에 결과를 내보내세요. "✅ 수학 (인강 1강) 완료 / 😄 92%" 한 줄짜리 자랑 카드.

SPEC Core Features · 핵심 기능

자세히 들여다보면.

F.01
요일별 학습 일정 등록
주간 반복 + 날짜별 인라인 달력 편집. 한 번 만들고 계속 굴립니다.
F.02
드래그앤드롭 순서 변경
세션 순서를 손가락 하나로 자유 변경. 오늘의 흐름에 맞춰 재배치.
F.03
타이머 + 자기평가
시간을 재고, 끝나면 😫 / 😐 / 😄 로 만족도를 남깁니다.
F.04
5열 도장 그리드
완료 세션이 도장처럼 쌓입니다. 채워가는 만족감.
F.05
무지개 월별 달력
요일별 7색 × 점수 5단계 오퍼시티로 한 달이 한 장에.
F.06
마이페이지 통계 + 추세
7 / 14 / 28일 막대 차트로 흐름을 봅니다.
F.07
