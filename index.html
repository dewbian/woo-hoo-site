도장(stamp) 마이크로 인터랙션 — “해냈다”는 감각의 시각화.

체크가 끝난 순간 종이에 도장이 찍히는 듯한 0.4초의 모션을 추가. 햅틱 + 옅은 효과음 + 살짝의 종이 그림자. 학생 5명 테스트 결과 “한 번 더 하고 싶다” 는 반응이 명확히 늘었다.

make-money — 같은 뉴스, 다른 페르소나 5종을 동시 생성.

하나의 원문 뉴스를 “ 경제 애널리스트 / 데이터 정리광 / 따뜻한 에세이스트 / 회의적 분석가 / 학부모 / Z세대 ” 의 시점으로 분기 생성하는 프롬프트 체인을 안정화. 티스토리 자동 발행은 30분 간격 큐로 분산.

googSky — “안정·적정·도전” 의 경계, 그 어딘가.

대학별 컷의 분산이 클수록 “적정” 의 범위가 모호해진다. 절대 등급 차이가 아니라 “안정성 점수” 로 다시 정의 중. 학부모 인터뷰 8건에서 가장 흔한 질문은 “그래서 진짜 안전한가요?” 였다.
