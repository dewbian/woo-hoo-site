# Play Store 등록 카피 (v0.4.1, 2026-05-19)

## 앱 이름
셀프체크 스터디

## 짧은 설명 (80자 이내)
체크리스트로 시작해 도장으로 쌓이고 카톡으로 자랑하는 학습 일기 앱.

## 자세한 설명 (Play Store)

> 공부, 기록, 자랑.
> 중·고등학생을 위한 오프라인 셀프체크 스터디 앱.

**■ 점수가 쌓이는 학습 일기**
"했다 / 안 했다"는 너무 거칠죠. 셀프체크 스터디는 목표 시간 대비 도달률(70%)과 만족도 이모티(30%)의 가중평균으로 세션 점수를 산출합니다. 같은 "완료"도 다르게 기록됩니다.

**■ 한 달이 한 장의 풍경으로**
요일별 무지개 7색 × 점수 5단계 오퍼시티의 월별 달력. 빈 셀이 적은 달, 진한 셀이 많은 주가 한눈에 보입니다.

**■ 진짜 보상은 카톡에서**
앱 안의 점수보다 부모·조부모·친구의 한 마디가 더 강력합니다. 시스템 공유 시트로 카톡에 결과를 내보내세요. ✅ 수학 (인강 1강) 완료 / 😄 92% — 한 줄짜리 자랑 카드.

**■ 핵심 기능**
- 요일별 학습 일정 등록 (주간 반복 + 날짜별 인라인 달력 편집)
- 드래그앤드롭으로 세션 순서 자유 변경
- 타이머로 시간 측정 + 자기평가 모달 (😫 / 😐 / 😄)
- 5열 도장 그리드 + 월별 무지개 달력
- 마이페이지에서 누적 통계 + 7/14/28일 막대 추세 차트
- 시작 / 자유시간 안내 알림 (진동·배너, 오프라인 로컬 알림)
- 3 진입점에서 카톡 공유 (Today / 스탬프 오늘 / 달력 셀)

**■ 개인정보**
- 완전 오프라인 작동, 모든 데이터는 사용자 기기에만 저장
- 광고 식별자 / 사용 기록 / 앱 로그 외부 전송 없음
- FCM 등 외부 푸시 서버 미사용 (로컬 알림만)
- 회원가입 없음

**■ 권장 환경**
Android 5.0 (Lollipop, API 21) 이상
