뉴스 자동 큐레이션·블로그 자동 발행 도구 · 개발 중

개인정보 처리방침.

make-money 는 사용자가 보유한 외부 블로그(티스토리 등)에 자동으로 콘텐츠를 등록하기 위한 운영자용 도구입니다. 따라서 본 서비스가 다루는 정보는 일반 소비자 정보가 아니라 운영자의 외부 서비스 접속 토큰과 발행 이력이 중심입니다.

개인정보의 수집 및 이용 목적

  • 서비스 기능 제공: 외부 블로그 자동 로그인(OAuth), 콘텐츠 자동 발행, 발행 결과 확인.
  • 서비스 개선: 자동 발행 실패율 모니터링, 오류 분석.
  • 요금 및 정산: 외부 광고 수익이 발생한 경우, 사용자 본인 확인을 위한 식별 정보.

수집하는 개인정보의 항목

  • 외부 서비스 인증 정보 (필수): 티스토리·구글 등 사용자가 직접 연결한 플랫폼의 OAuth 액세스 토큰. 이 토큰은 암호화되어 저장되며, 자동 발행을 위해서만 사용됩니다.
  • 발행 이력 (필수): 자동 등록된 글의 URL, 제목, 발행 시각, 성공/실패 상태.
  • 이메일 (선택): 알림·뉴스레터 수신을 동의한 경우에 한해 보관.
  • 자동 수집 정보: 기기 식별자, IP, 앱 로그, 오류 데이터.

본 서비스는 운영자가 새로 작성한 콘텐츠를 다룰 뿐, 운영자의 블로그에 이미 존재하던 기존 글이나 방문자 데이터는 수집하지 않습니다.

개인정보의 보유 및 이용 기간

  • OAuth 액세스 토큰: 사용자가 외부 서비스 연결을 해제하거나 앱을 삭제하는 즉시 파기됩니다.
  • 발행 이력: 사용자가 직접 삭제하거나, 계정 종료 후 30일이 지나면 파기됩니다.
  • 앱 로그 및 성능 데이터: 관련 법령에 따라 일정 기간 보관 후 파기됩니다.

제3자에게의 개인정보 제공 및 위탁

본 서비스는 외부 자동 발행을 목적으로 다음 플랫폼의 API 를 호출합니다. 각 API 호출에 사용되는 인증 정보는 사용자가 명시적으로 연결한 경우에만 사용됩니다.

  • 티스토리 OpenAPI: 사용자 본인의 블로그에 글을 발행하기 위해 호출.
  • 대형 언어 모델(LLM) 제공사: 뉴스 요약·관점별 콘텐츠 생성을 위해 호출. 운영자의 식별 정보는 전송하지 않으며, 뉴스 본문과 프롬프트만 전송됩니다.
  • 구글 푸시 알림(FCM): 발행 완료/실패 알림을 위한 기술 서비스.

위 외의 제3자에게 사용자의 개인정보를 양도·판매하지 않습니다.

  • 앱을 삭제하면 기기에 저장된 모든 로컬 데이터가 즉시 파기됩니다.
